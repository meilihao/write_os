# write os
参照<<一个64位操作系统的设计与实现>>写一个64 bit的os.

目的:
1. 理解os
1. just for fun!

模拟器选择qemu, 兼容性更好, 更容易发现问题, 调试可针对场景bochs/qemu一起上.
